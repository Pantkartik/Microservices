# Middleware 

A Middleware is like a middle layer which act like a parser between the request response manages the actions

# Steps for middleware

1. Initial response ware 

  In this the user (client) makes the request anthing like get,post,patch,delete to the server.

2. Middleware Phase
  In this phase the predesigned or user defined middleware act as a regualtor barrier b/w the response and the request input 

  It control whether the request is valid and how will be the flow of request response helps in delaying the output if it needed to to prevent any attacks

3. Final Phase

   This is the final destination phase where the response is executed and displayed back to client from server
